## Technologies
Project is created with:
* Node: 9.1.1
* React: 18.2.0
* Recharts: 2.1.16
* Axios: 1.2.1


## Prérequis
* Installing NodeJS

## Installation 
1. Launch the backend
2. Clone repository 
```
git clone https://github.com/HaydenLeDev/sport_see.git
```
3. Install npm packages
```
npm i
```
4. Start the project
```
npm start
```
